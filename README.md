# Kartik Jethva Portfolio

This is a static portfolio website generated from resume content.

## Files
- `index.html` – Main portfolio page
- `styles.css` – Styling for the portfolio
- `script.js` – Mobile menu behavior
- `Kartik_Resume.pdf` – Resume file linked from the portfolio
- `resume_text.txt` – Extracted text from the resume PDF

## Usage
1. Open `index.html` in a browser.
2. Or run the built-in server:
   - `python server.py`
   - the site will open at `http://127.0.0.1:8000`

## Notes
The portfolio includes sections for summary, skills, experience, projects, education, and contact details.
